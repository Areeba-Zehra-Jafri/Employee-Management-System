# Employee Management System

A simple **Employee Management System** developed in **C** as a first-semester programming project. The application provides a menu-driven interface to manage employee records, perform basic HR operations, and demonstrate fundamental C programming concepts such as file handling, structures, functions, and dynamic memory allocation.

---

## Features

* Employee login system
* Add employee records
* View all employee records
* Search employees by ID
* Edit employee information
* Delete employee records
* Attendance tracking
* Salary calculator with attendance and overtime adjustments
* Employee notifications
* Company information section
* Persistent data storage using files

---

## Technologies Used

* C Programming
* File Handling
* Structures
* Functions
* Dynamic Memory Allocation
* Windows Console API

## Employee Information Stored

Each employee record contains:

* Employee ID
* Name
* Gender
* Age
* Salary
* Designation
* Address
* Marital Status
* Date of Birth
* CNIC Number
* Attendance

## How to Run

### Compile

```bash
gcc project.c -o project
```

### Run

On Windows:

```bash
project.exe
```

Or compile and execute using any C compiler that supports the Windows-specific libraries used in the project.
